## 给药途径说明

| 缩写 | 全称 | 说明 |
|------|------|------|
| PO | 经口 | 口服，混饮或直接喂服 |
| IM | 肌内注射 | 胸肌或腿部肌注 |
| IV | 静脉注射 | 颈静脉或翼静脉 |
| SC | 皮下注射 | 颈背部皮下 |
| ICe | 体腔注射 | 腹腔或气囊内 |
| IT | 气管内 | 雾化或气管滴注 |
| IO | 骨内 | 骨髓腔内 |

## 剂量频次说明

| 缩写 | 含义 |
|------|------|
| q8h | 每 8 小时一次 |
| q12h | 每 12 小时一次（一天两次） |
| q24h | 每 24 小时一次（一天一次） |
| q48h | 每 48 小时一次（两天一次） |
| SID | 每日一次 |
| BID | 每日两次 |
| TID | 每日三次 |

> **注：** 以上剂量为参考值，具体剂量需结合鸟种、病情、体重综合判断。
> **PD** = 药效学指导剂量，需要监测血药浓度。
