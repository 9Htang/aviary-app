## 数据流

```
用户输入 (鸟种 + 体重 + 药品)
       ↓
  剂量计算引擎 ← 药典数据库 (JSON)
       ↓
  计算结果 (用量 / 浓度 / 频次)
       ↓
  显示结果 + 可选保存记录
```

## 核心模块

### Models（数据模型）
- `Drug` — 药品对象（名称、分类、剂量范围、途径、频次、适用鸟种）
- `DoseRecord` — 用药记录（药品、剂量、鸟种、体重、时间戳）

### Data（药典数据）
- 内置 JSON 数据库，打包在 APK 中
- 后续可支持从云端/本地文件导入更新

### Screens（界面）
- 首页 — 快速计算入口
- 计算页 — 输入参数 → 输出结果
- 药典浏览 — 按分类/名称浏览所有药品
- 历史记录 — 查看以往计算记录

### Utils（工具）
- 剂量计算器 — 核心算法
- 单位换算 — g/kg/mg/µg 互转
