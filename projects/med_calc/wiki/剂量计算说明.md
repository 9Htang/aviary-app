# 🧮 剂量计算说明

## 计算公式

### 1. 按体重给药（最常见）

```
用药量 (mg) = 体重 (kg) × 推荐剂量 (mg/kg)
```

**示例：**
> 一只 **玄凤鹦鹉** 体重 **100g**，使用 **多西环素** 剂量 **25 mg/kg**：
> 100g = 0.1 kg
> 用药量 = 0.1 × 25 = **2.5 mg**

### 2. 混饮给药

```
药物浓度 (mg/L) = 推荐混饮浓度
实际配水量 (L) = 日饮水量 × 疗程天数
```

**示例：**
> 推荐 **500 mg/L** 混饮，2 只鹦鹉 3 天疗程，每日约饮 50 mL：
> 总水量 = 0.05 L × 3天 = 0.15 L
> 总药量 = 500 × 0.15 = **75 mg**

### 3. 饲料添加

```
药物添加量 (mg) = 饲料量 (kg) × 推荐添加量 (mg/kg 饲料)
```

## 单位换算

| 单位 | 换算 |
|------|------|
| 1 kg | = 1000 g |
| 1 g | = 1000 mg |
| 1 mg | = 1000 µg |
| 1 L | = 1000 mL |

## 用药安全注意事项

1. **按体重精确计算** — 过量可能导致中毒
2. **注意给药途径** — 不同途径吸收率不同
3. **注意给药频次** — 遵从药代动力学规律
4. **疗程完整性** — 即使症状缓解也应完成疗程
5. **药物相互作用** — 某些药物不能联用
6. **肝肾功能** — 肝肾疾病需调整剂量
