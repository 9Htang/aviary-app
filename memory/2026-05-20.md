## 重大事件

### 鸟舍管理系统建立
小豆要搭建本地鹦鹉鸟舍管理系统，技术栈为 **SQLite + Express.js + EJS**。
数据库文件：`aviary/aviary.db`，网页服务：`aviary/server.js`
访问地址：http://127.0.0.1:3456 （局域网手机 http://192.168.10.9:3456）

### 数据库表结构
- species（品种）
- birds（个体鸟：含 father_id/mother_id 自引用谱系、ring_number UNIQUE、status、breeding_status、source、image_path）
- mutations（基因名 + 遗传类型）
- bird_mutations（个体基因型）
- pairings（配对记录，含 start_date/status/male_id/female_id）
- breeding_events / offspring / relationship_cache

### 配对状态（最终版）
未配对 / 不可配对 / 已配对

### 来源（最终版）
外来 / 自繁育 / 未知

### 遗传类型体系
显性遗传 / 隐性遗传 / 性联遗传 / 不完全显性 / 多基因遗传 / 未知

### 牡丹鹦鹉基因库（18项，来自 CSV）
- 显性：派特、花桃
- 不完全显性：金顶、紫罗兰、墨色
- 隐性：百趴、深华、浅华、边化、金面、白面、蓝色、松石色、浅绿色
- 性联：闪光、肉桂、黄化
- 多基因：长羽

### 网站功能
- 卡片式首页（含加载更多、图片懒加载）
- 按 ID/脚环/名字 搜索
- 详情页：信息、基因、子女、配对历史、当前配偶
- 新增鸟只：品种选单、基因勾选（按品种分组）、日期选择器、性别自动切换性联基因选项
- 图片裁切上传（CropperJS 3:4 竖版，最长边 400px，JPEG 60%）
- 编辑鸟只：修改配对状态时自动弹窗选配偶
- 年龄自动计算（8个月以下/4年以上 → 不宜配对，红色标记）
- 编辑页返回首页自动刷新（no-store + unload 防止 bfcache）
