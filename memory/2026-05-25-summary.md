**移动端 Web 适配（核心改造）：**
- ❌ 暴力缩小（65/80px 图片）→ 布局溢出
- ❌ column-reverse → 照片挤压信息
- ❌ 多个琐碎 media query → 冲突
- ✅ **≤768px 单一断点，流式卡片布局**
  - 照片全宽展示 (max-width 260px)

  - 名字+操作按钮分离：名字一行，按钮一行
  - 信息卡片左右分栏，`space-between` 对齐

### 无效步骤记录（避免重复）

1. **`edit` 工具多行替换** — 文件有 `\r\n` 导致 `oldText` 匹配失败，改用 Node.js 脚本写文件
2. **`PowerShell` 内联 `node -e`** — `$` 和 `"` 被 PowerShell 解释器劫持，改写入 `.js` 文件再执行
3. **`const SnackBar(...)` 运行时变量** — Dart 不允许 `const` 嵌套非 `const` 表达式
4. **按钮 `Expanded` 包裹** — 多次尝试失败，直接设置相同 `minimumSize` 更可靠
5. **`showMenu()` 多选** — 每次选择关闭菜单，改为 `AlertDialog + CheckboxListTile`
6. **CSS 多断点** — 600/420/400 多个碎断点互相覆盖，改为单一 768px
