### 💀 重大教训
1. `taskkill /f /im node.exe` → 杀死 OpenClaw 自己（AGENTS.md 警告过）
2. 50+ 旧 node 进程堆积 → 新进程因端口被占静默失败 → 测试全部指向旧代码
3. GBK→UTF-8 混写导致文件损坏，修复耗费50+轮次
4. PowerShell `node -e "中文代码"` 必崩 → 改用 `write` + `node file.js`
5. `Out-File -Encoding UTF8` 在 PS5.1 可能写出 UTF-16 → 用 `edit` 工具替代
